2021华东师大毕业流程 全记录

学位审批论文结束，就可以启动离校申请了。在师生综合平台上可以看到“离校手续”（访问离校页面需要连入校内网）。

在提交离校申请之后，可以看到办理状态，其中“已办理手续部门”“未办理手续部门”与“办理未通过部门”将会随着办理进度而改变。此状态也可以在企业微信->工作台-> 学生服务 -> 毕业离校信息 -> 离校查询看到办理进度。

我的系统中办手续涉及到的部门有：

  • 研究生院( ),
  • 助学贷款办公室( ),
  • 图书馆(图书馆自助离校),
  • 财务处( ),
  • 就业咨询服务中心( ),
  • 保卫处( )，
  • 档案馆,
  • 后勤保障部
  • 公房综合治理办公室
  • 数据科学与工程学院
  • 校园卡中心

其中，下述部门的手续是会自动通过的：

  • 研究生院( ),
  • 助学贷款办公室( ),
  • 财务处( ),
  • 保卫处( )，

剩余各部门的办理流程分别为：

图书馆(图书馆自助离校)

  • 办理时长：约3分钟

首先访问图书馆主页：http://www.lib.ecnu.edu.cn/，进入学位论文提交页面

根据自己的论文情况将相应内容填入表单，然后上传论文PDF即可。

接下来就打印一本论文交到图书馆前台，工作人员将会检查图书借还状态，并修改图书馆的权限等项目（应该就是不能再借书了）。

这时再到离校手续页面，在“图书馆自助离校办理”页面中，上方所有状态都是完成时，点击最后的“办理”按钮即可。

这样，图书馆离校手续就办理完成。

校园卡中心

  • 地点：信息化治理办公室（九舍牛奶棚）
  • 办理时长：约3分钟

位置在信息化治理办公室，在第9宿舍牛奶棚旁边，前台有“校园卡业务”，将校园卡交给工作人员，立等可取。其中校园卡中的余额如果较多，则会退到银行卡上，余额少的则现场会返还现金（最小单位为角）或者可以先在校园卡综合服务平台上将余额转到指定的银行账户，再办理注销校园卡业务。

这一步操作之后，校园卡的所有功能均失效，不能再进行刷卡消费、充值等一切动作，建议最后进行这一步。

就业咨询服务中心

  • 办理时长：约20分钟

先在就业信息网填写相应的就业信息。并 将相应材料（一般是签订的就业协议）交到大学生活动中心一楼就业咨询服务中心。

后勤保障部 & 公房综合治理办公室

  • 办理时长：不定，取决于物业审批速度

这一步是退宿舍，首先在微信企业号或企业微信上提交退宿申请（工作台-> 学生服务 -> 学生宿舍 -> 微服务 -> 退宿申请 ）。提交后需要等待辅导员与物业经理审批，辅导员审批一般比较快，物业经理审批不一定，有可能需要好几天。物业经理审核之后，就可以去宿舍楼门口将宿舍钥匙归还，在宿舍信息签字，等待宿管中心进行审批，审批后即流程通过。

离校手续单

按照要求，当“未办理手续部分”只剩学院与档案馆的时候，就可以凭离校手续单领取双证。离校手续单上需要完成的有归还设备、归还实验室钥匙、导师签字、学院盖章。

  • 归还设备：计算机楼找蒲鹏老师，将显示器、主机、鼠标、键盘搬到指定位置
  • 归还实验室钥匙：（若有）交给辅导员
  • 导师签字：导师签字
  • 学院盖章：数学馆东106

至此即可领取毕业证与学位证。
