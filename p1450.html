LaTeX的双栏布局与平行双栏对照排版

双栏布局

很多出版物都是按文字双栏进行布局，在LaTeX中可以用 \twocolumn 表示文档的双栏

平行双栏对照

在进行翻译文稿时，会需要左右分栏对照进行，在Word中常用的方法是建一个一行两列的大表格，将文字全部放入表格中进行对照。使用LaTeX则可使用parallel宏包实现这样双栏对照。

更详细的文档可以参考： https://mirror.bjtu.edu.cn/CTAN/macros/latex/contrib/parallel/parallel.pdf 

\documentclass{article}
\usepackage[a4paper,top=1.27cm,bottom=1.27cm,left=1.27cm,right=1.27cm,%
            includehead,includefoot]{geometry}
\usepackage[UTF8]{ctex}
\usepackage{parallel}
\begin{document}
\begin{Parallel}{0.45\textwidth}{0.45\textwidth}
    \ParallelLText{
    左侧文字1
    }
    \ParallelRText{
    右侧文字1
    }
    \ParallelPar
    \ParallelLText{
    左侧文字2
    }
    \ParallelRText{
    右侧文字2
    }
    \ParallelPar
\end{Parallel}
\end{document}

其中\ParallelLText与\ParallelRText分别是左右两栏进行对照的文字内容，\ParallelPar是一个分块符，用于对不同的区块进行分隔。如果需要在对照的两栏中间有竖线，则在\begin{Parallel}上添加选项[v]即可
